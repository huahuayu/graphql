# go graphql uniswap v2 example

This is an example demonstrate how to query uniswap v2 data by graphql in golang.

## usage

1. git clone 

```bash
git clone https://github.com/huahuayu/graphql.git
```

2. build

```bash
cd graphql
go build ./...
```

3. run

```bash
go run main.go
```
